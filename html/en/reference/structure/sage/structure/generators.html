class sage.structure.generators.Generators_list¶

Bases: sage.structure.generators.Generators_finite

This class represents a set of generators as a tuple of elements, indexed by the integers 0 to len(self)-1.

It is the easiest to use of all the generator classes, and gets constructed implicitly when a list is passed into code{_populate_generators_}.

get_from_index(i)¶
EXAMPLES:
sage: from sage.structure.generators import Generators_list sage: gens = Generators_list(ZZ, [5,9], Rings) sage: gens.get_from_index(0) 5
list()¶

Actually returns a tuple, which is immutable.

EXAMPLES:
sage: from sage.structure.generators import Generators_list sage: gens = Generators_list(ZZ, (1,2,3), Rings) sage: gens.list() (1, 2, 3)
